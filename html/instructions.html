LinkedIn Easy Apply Bot - 使用指南

LinkedIn Easy Apply Bot 是一个强大的自动化工具，可以帮助您在 LinkedIn 上自动搜索和申请工作。该工具支持自动填写申请表格、上传简历和求职信，并可以使用 AI 自动生成回答面试问题。

功能特点

🔍

自动搜索工作

根据您设置的职位、地点和筛选条件自动搜索 LinkedIn 上的工作机会。

📝

自动填写申请表格

自动填写个人信息、工作经验、教育背景等申请表格内容，大大节省申请时间。

📄

上传简历和求职信

自动上传您的简历和求职信（如果需要）。

🤖

AI 回答生成

支持使用云服务 API 自动生成对申请问题的回答。

⚖️

职位匹配度评估

使用 AI 分析职位描述和您的资历，评估职位是否值得申请。

📊

应用记录

记录已申请的职位，方便后续跟踪。

安装和准备

系统要求

  • Windows 操作系统（批处理文件基于 Windows 编写）
  • Python 3.8 或更高版本 下载python
  • Chrome 浏览器
  • LinkedIn 账号

环境准备

  1. 确保已安装 Python 和 pip
  2. 确保有可用的 LinkedIn 账号
  3. 准备好您的简历 PDF 文件
  4. 如果使用 OpenAI API，准备好 API 密钥

使用批处理文件

start.bat 文件

start.bat 是主要的启动脚本，用于启动图形界面配置工具。使用方法：

  1. 在第一次使用时双击 setup.bat 安装程序必备的依赖包
  2. 双击 start.bat 文件
  3. 等待程序启动
  4. 在图形界面中配置您的设置

注意： 首次运行时，如果没有 config.yaml 文件，系统会提示您创建一个配置文件。

运行机器人

配置完成后，您可以选择：

  • 通过图形界面-操作与状态中启动机器人（推荐新用户使用）

配置说明

基本设置

设置项 说明
LinkedIn 邮箱 您的 LinkedIn 账号邮箱
LinkedIn 密码 您的 LinkedIn 账号密码
OpenAI API 密钥 可选，提供后可使用 AI 功能，不提供时将使用云服务 API
使用云服务 API 是否使用云服务 API（默认启用）
简历路径 您的简历 PDF 文件路径
简历文本路径 您的简历文本文件路径，用于AI匹配问题，可直接将pdf中的文本内容复制到txt文件，之后上传该txt路径
求职信路径 可选，您的求职信文件路径

搜索设置

设置项 说明
职位关键词 要搜索的职位关键词，如 "Software Engineer"
地点 搜索的地理位置，如 "New York"
搜索距离 从中心位置搜索的距离（单位：英里）
远程工作 是否只搜索远程工作
最新发布优先 是否优先显示最新发布的职位

个人信息

在此部分填写您的详细个人信息，这些信息将用于自动填写申请表格：

  • 姓名、性别、年龄
  • 联系方式
  • 地址信息
  • 自我介绍

工作经验

添加您的工作经验，包括：

  • 公司名称
  • 职位名称
  • 工作时间（年月）
  • 工作描述
  • 是否为当前工作

教育背景

添加您的教育经历，包括：

  • 学校名称
  • 学位
  • 专业
  • 就读时间（年月）
  • 是否在读

AI 功能说明

云服务 API 使用说明

系统使用云服务 API 处理申请问题和职位评估：

  • 使用默认密钥：不提供 OpenAI API 密钥时，系统将使用云服务的默认 API 密钥
  • 使用自定义密钥：提供自己的 OpenAI API 密钥后，云服务将使用您的密钥进行请求

当您在设置中填写了自己的 OpenAI API 密钥时，系统会将此密钥传递给云服务，云服务将优先使用您的密钥来处理请求。这样可以确保请求使用您自己的账户额度。

AI 答题和职位评估

AI 服务会帮助：

  • 自动生成申请问题的回答
  • 评估职位是否与您的资历匹配

常见问题

Q: 机器人无法登录 LinkedIn 怎么办？

A: 可能是因为 LinkedIn 要求安全验证。程序会提示您手动完成验证，然后继续运行。

Q: 为什么有些字段没有自动填写？

A: 某些特殊或不常见的字段可能无法自动识别。您可以在 "unprepared_questions.csv" 日志中查看这些字段，并在下次运行前添加自定义答案。

Q: AI 生成的回答不够准确怎么办？

A: 您可以通过配置自定义问题和答案来覆盖 AI 的回答。

Q: 如何限制机器人只申请特定类型的工作？

A: 您可以在配置中设置黑名单（公司、职位、发布者）和使用职位匹配度评估功能。
